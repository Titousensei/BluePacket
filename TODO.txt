=== Features ===
- list of native type with anonymous single field struct
- xbool -> bits in a byte
- xshort: variable length short, mostly small.
  - start with a byte, if value == -128 read a short.
- xint: variable length int, mostly small.
  - start with a byte, if value == -128 read a short, if value == -127 read an int.
- java convenience setters DONE
  Example:
  - setField(Type[] value)  // current version
  - setField(Type... value)  // replace current version
  - setField(Collection<Type> value) // trivial toArray conversion
